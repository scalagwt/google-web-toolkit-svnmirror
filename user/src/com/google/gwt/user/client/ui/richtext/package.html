Rich text editing, Highlights, and their associated classes. This package contains a rich text widget {@link com.google.gwt.user.client.ui.richtext.RichTextArea}, which is an analog of the {@com.google.gwt.user.client.ui.TextArea} class which can edit rich text. Also included is a {@link com.google.gwt.user.client.ui.richtext.RichTextEditor} widget, which provides the UI needed to actually input HTML. Spell checking integration is also included.

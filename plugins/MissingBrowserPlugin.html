No GWT Browser Plugin Detected

GWT Development Mode needs a browser plugin to operate. Please install the appropriate one for your browser.
  • Firefox
    Install the appropriate Firefox plugin via the normal plugin install process.
    • Version 1.5-2.0
    • Version 3.0
    • Version 3.0 alternate libraries (try this if the one above doesn't work on Linux, known to be needed on Fedora Core 10))
    • Version 3.5+

  • Safari 3/4 on Mac/OSX
    • Run the installer from this disk image.

  • Internet Explorer on Windows (32-bit IE)
    • Download oophm.dll to local disk, then run regsvr32 oophm.dll and restart IE.

Troubleshooting

If you believe you have the plugin installed in your current browser (and profile, in the case of Firefox), see the TroubleshootingOOPHM wiki page.
